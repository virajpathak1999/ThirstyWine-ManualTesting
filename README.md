# ThirstyWine-ManualTesting
# 🍷 Thirsty Wine – E-Commerce Web App Testing

## 📌 Project Overview
Thirsty Wine is an e-commerce web application where users can browse, search, purchase, and review different wines.  

This project focuses on **manual testing of the frontend**, ensuring that all core features work correctly, are user-friendly, and perform well across different browsers and devices.
